1- Display

Ici nous voyons comment réagit un container grid, grid va nous servir à gérer l'alignement et le positionnement de nos boxs dans les boxs et comprendre l'héritage parent => enfant

Comme son nom l'indique grid va nous permetre de réaliser une grille !

Display grid

Ici on gère notre container avec "display: grid" seulement les espaces entre nos box grâce à:

- "grid-row-gap: 10px;" pour l'inter-espècement entre les rangées (row)

- "grid-column-gap: 10px;" pour l'inter-espècement entre les columns (column) (que nous ne pouvons pas voir ici)

Grid template

Ici on gère nos espacement en faisant une structure de base comprenant 3 column

On utilise "grid-template-columns: 1fr 3fr 1fr;" ce qui divise notre page en 3 column proportionel

- La première fait 1fr donc 1 x 1

- La deuxième fait 3fr donc 3 x 1

- La troisieme fait 1fr donc 1 x 1
