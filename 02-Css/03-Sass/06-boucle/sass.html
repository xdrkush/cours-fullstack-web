06 Les Boucles

Pour les boucles c'est pareil que les fonctions, le principe est le même que en javascript donc vous allez profiter du language CSS afin de comprendre vos premières mécanique logique.

Le But est d'éviter les répétitions afin que notre code soit plus facile à écrire mais que le résultat de notre css soit le même (sans les répétition)

Petit cas pratique ?

Donc ici on voit que l'on a 2 boucle, 1 pour les padding Y et 1 pour les margin X

Mais concrètement, on va écrire que 3 ligne de SASS pour avoir 25 classe de creer

Boucle SASS


    // Padding de l'axe Y (haut et bas)
    @for $i from 1 to (25 + 1)
        // On declare notre classe
        .py-#{$i}
            // et le comportement de la classe
            padding: #{$i}px 0 #{$i}px 0
                    
    // Margin de l'axe X (gauche é droite)
    @for $i from 1 to (25 + 1)
        // On declare notre classe
        .mx-#{$i}
            // et le comportement de la classe
            padding: 0 #{$i}px 0 #{$i}px

Ici on rajoute de la logique au container

la function col-x


        // On definit le nombre de column de notre width total
        $column-count: 12
        // On declare notre fonction qui prend en parametre un nombre
        @function column-width($num)
            // On retourne la valeur de la taille de notre container
            @return $num * 100% / $column-count
                    
        // Ici notre boucle va faire 12 tour
        @for $i from 1 to ($column-count + 1)
        // On parametre le debut de notre classe avec le $i
            .col-#{$i}
            // On utilise la fonction dans notre boucle
            width: column-width($i)

On pourrait aussi jouer avec image ?

Ici on joue avec le nom des classe pour allez chercher nos images, mais il existe aussi d'autre logique, (key: value, key2: value2, ...) par exemple, ici on utilise la première méthode

Voici la doc pour la second solution, >> ICI <<

Le SASS


    // Ici est creer notre liste d'item
    $list: arrr btc doge eth kmd
    // Notre boucle avec les élément de la liste
    @each $i in $list
        // notre classe à creer avec la liste
        .img-#{$i}
            height: 500px
            width: 500px
            background: no-repeat
            // on va chercher l'image avec le même nom que l'index de la boucle
            background-image: url("../images/#{$i}.png")
