02- Les imports

Comment sa fonctionne ?

Les import en sass sont très utile afin de rendre nos fichiers et notre architecture plus lisible et simple à ce retrouver

Plutot que d'avoir un fichier de 1000 ligne de css on va pouvoir de retrouver avec plusieurs fichier avec un role bien définit

Voici comment on écrit un import "@import lefichier"

Par exemple si vous regarder dans l'architecture on va ce retrouver grâce à des fichier au nom explicite:

  • index.sass Qui va nous permetre de centraliser nos ressources. (Bootstrap est un bon exemple )
  • variables.sass Qui va nous permettre de stocker nos variables
  • border.sass Ici les bordure
  • background.sass les background
  • ...

Attention Il y a des ordres de lecture de notre code à respecter:

à ne pas faire:

 @import border // à besoin de variable mais ne peux pas le reconnaitre vu que variable sera lu après 
 @import variable // variable sera lu après border

à faire:

 @import variable // on définit les variable 
 @import border // ensuite les variable peuvent etre utiliser dans border

par exemple si on definit un @import variable après un @import border par exemple, vu que @import border contient des variables présentes dans @import variable votre code sera bugué !
