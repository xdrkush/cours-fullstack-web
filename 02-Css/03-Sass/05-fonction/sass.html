Comment sa marche ?

Les fonctions marche sur le même principe quand JS (javascript) bon si vous en êtes ici c'est que vous n'avez peut être pas encore vue JS

Ce n'est pas très compliquer, dans un premier tant on déclare notre fonction avec "@function nomDeLaFunction(parametre1, parametre2)"

Ensuite on va lui demander de realiser une tache avec nos parametre parametre1 * parametre2

d'ailleur on pourrait stocker la valeur de la multiplication dans une variable $result: parametre1 * parametre2

Pour finit on demandera à notre fonction de retourner un résultat que l'on pourrait écrire de plusieurs manières

Solution 1: "return parametre1 * parametre2" ou Solution 2: "return $result"

Ce qui donnerais une fonctions de ce type

SASS


            @function nomDeLaFonction($params1, $params2)
                $result: $params1 * $params2
                @return $result

Petit cas pratique ?

Vous avez certainement remarquer que les couleurs avait changer depuis le dernier tuto et bien cela à été fait avec un fonction sass

le Body

    
      // La fonction mix est native dans sass, elle nous permet de faire une moyenne
      // 60% de $primary-color // 40% de $secondary-color

      body
          background-color: mix($primary-color, $secondary-color, 60%)

le Header

    
      // On définit notre fonction qui prend 2 argument (la couleur, et l'instensité)
      @function inverted($color, $amount: 100%)
          // On viens définir la variable de notre nouvelle couleur
          $inverse: change-color($color, $hue: hue($color) + 180)
          // Et on retourne notre couleur (le return sera la valeur de la fonction)
          @return mix($inverse, $color, $amount)
                    
      // On definit notre header
      header
          // La couleur utiliser sera l'inverse de $primary-color
          background-color: inverted($primary-color, 80%)

la function col-x

    
        // On definit le nombre de column de notre width total
        $column-count: 12
        // On declare notre fonction qui prend en parametre un nombre
        @function column-width($num)
            // On retourne la valeur de la taille de notre container
            @return $num * 100% / $column-count

        .col-2
            width: column-width(2)
