Comment installer sass ?

Biensur vous devez avoir node et npm d'installer sur votre machine pour suivre la commande si dessous

sudo npm -g sass

Comment compiler un ou plusieurs fichiers sass ?

Vous avez 2 manière principal de compiler votre sass, soit en ligne de commande (se que nous allons faire), ou avec des petites extensions du genre sass-compiler sur visual studio code, ou autre, sinon avec des architectures plus complexe du style avec webpack ce que nous aborderont pas ici !

Pour un seul fichiers il va falloir un fichier .sass ou .scss

Par exemple pour se dossier ce rendre dans le dossier sass et executer la commande suivante

sass --watch index.sass:style.scss

Se qui va vous permettre d'avoir le fichier index.sass en entrée (input) et de vous sortir un fichier style.css (output)

Pour compiler plusieurs fichier sass/scss dans un seul fichier css vous allez juste devoir importer vos fichiers sass dans un seul et tout ce compilera depuis le fichier mère. (nous aborderont l'import de l'exercice suivant)

par exemple j'ai un fichier index.sass et un fichier variable.sass

alors je vais importer le fichier variable.sass dans le fichier index.sass ce qui nous laissera l'opportunité de compiler le fichier index.sass seulement vu qui importera le fichier variable.sass et nous sortira que un seul fichier style.css

01 Les variables

Vous pouvez dès maintenant allez voir dans le fichier index.sass pour voir comment sont définit les variable et comment les utiliser

On définit une variable ainsi "$green: #026782"

Et on l'utilise ainsi "color: $green"

On peux biensur faire utiliser une variable à une variable "$primary-color: $green"
