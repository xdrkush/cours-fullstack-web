Les nestings

Comment sa marche ?

Les nestings en sass on la meme propriétié que en css classque il n'y a que la façon de l'écrire qui est différente

Par exemple:

SASS


                .parent
                    color: $blue
                    .child
                        color: $purple

                .child
                    color: $red

CSS


                    .parent {
                        color: #4B65E9;
                    }
                    .parent .child {
                        color: #710F77;
                    }
                    .child {
                        color: #E32020;
                    }

Ce qui nous donne:

Mon text parent !

Mon text enfant dans le parent !

Mon text enfant seul !
